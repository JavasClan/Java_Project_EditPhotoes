## 一、项目概述
本项目是一款**桌面端可视化图片处理工具**，具备清晰的UI界面布局（菜单栏+工具栏+预览区+参数区+状态区），用户无需掌握专业技术，仅通过点击、输入、选择等简单UI操作，即可完成从基础图片编辑到AI增强的全流程处理。项目支持本地图片加载与保存，操作过程实时预览，且提供完善的操作回退/重做机制，兼顾易用性与功能性，适用于普通用户日常图片美化、修复及创意制作场景。

## 三、UI可调用的核心功能（含操作流程+参数配置）
这是撰写用户说明书的核心内容，需明确每个功能在UI上的操作路径、可配置参数及呈现效果：

### 1. 基础图片处理功能（UI优先展示，操作最便捷）
#### （1）图片加载与保存
- UI操作路径：
    1. 加载：顶部菜单栏「文件」→「打开图片」，弹出本地文件选择窗口，选中图片后点击“确定”，图片自动显示在中央预览区
    2. 保存：顶部菜单栏「文件」→「保存图片」（覆盖原图）/「另存为图片」（自定义保存路径+文件名），默认保存格式为PNG，支持下拉选择JPG/BMP
    3. 批量导入：顶部菜单栏「编辑」→「批量操作导入」，可选中多个图片，导入后按顺序在预览区切换显示
- UI反馈：加载完成后状态区显示“图片加载成功：XXX（尺寸：W×H，大小：XX MB）”；保存成功后显示“图片已保存至：XXX路径”

#### （2）旋转功能
- UI操作路径：左侧工具栏「旋转图标」（箭头循环样式）→ 右侧参数区选择旋转角度（90°/180°/270°/自定义角度）→ 点击「应用」
- 可配置参数：
    - 预设角度：90°（顺时针）、180°、270°（顺时针，下拉选择）
    - 自定义角度：0°~360°（输入框+滑动条调节，精度0.1°）
- UI反馈：调节参数时实时预览旋转效果，点击“应用”后状态区显示“旋转操作成功，角度：XX°”，支持撤销/重做

#### （3）裁剪功能
- UI操作路径：
    1. 左侧工具栏点击「裁剪图标」（矩形框样式），预览区图片出现可拖拽虚线裁剪框
    2. 调节方式： 鼠标拖拽裁剪框四角/边调整大小；
    3. 鼠标抬起自动完成裁剪
- UI反馈：拖拽裁剪框时实时显示裁剪区域尺寸，参数输入错误（如宽高超出图片范围）时，参数区边框变红，状态区提示“裁剪参数无效，请重新设置”

#### （4）撤销/重做/恢复原图
- UI操作路径：
    - 撤销：顶部菜单栏「编辑」→「撤销」/ 快捷键Ctrl+Z / 工具栏撤销图标（向左箭头）
    - 重做：顶部菜单栏「编辑」→「重做」/ 快捷键Ctrl+Y / 工具栏重做图标（向右箭头）
    - 恢复原图：顶部菜单栏「编辑」→「恢复原图」/ 工具栏原图图标（图片样式）
- UI限制与反馈：最多保留10条操作记录，无可撤销操作时，撤销按钮变灰（不可点击），状态区提示“已无历史操作可撤销”；恢复原图后，所有操作记录清空，状态区提示“已恢复至图片初始状态”

#### （5）翻转/镜像功能
- UI操作路径：左侧工具栏「翻转/镜像组」→ 选择对应功能（水平翻转/垂直翻转/水平镜像/垂直镜像）→ 右侧参数区配置（仅镜像需配置）→ 点击「应用」
- 可配置参数（仅镜像功能）：
    - 水平镜像分割比例：0.1~0.9（默认0.5，即中间分割，滑动条调节）
    - 垂直镜像分割比例：0.1~0.9（默认0.5，滑动条调节）
- UI反馈：点击功能图标后实时预览效果，镜像比例调节时，预览区同步显示分割线及镜像效果

### 2. 扩展图片处理功能（UI分组折叠，按需展开）
#### （1）色彩调整功能
- UI操作路径：左侧工具栏「扩展功能」→ 展开「色彩调整」→ 选择子功能（亮度/对比度/灰度化）→ 右侧参数区调节参数 → 点击「应用」
- 各子功能可配置参数：
    - 亮度：-50% ~ +50%（默认0，滑动条+输入框，负值降低亮度，正值提高亮度）
    - 对比度：-50% ~ +50%（默认0，滑动条+输入框，负值降低对比度，正值提高对比度）
    - 灰度化：下拉选择方式（亮度加权法/平均值法，默认亮度加权法，无需额外参数）
- UI反馈：调节参数时实时预览色彩变化，参数超出范围时，输入框自动修正为边界值，状态区提示“参数已自动调整至有效范围”

#### （2）几何缩放功能
- UI操作路径：左侧工具栏「扩展功能」→ 「缩放」→ 右侧参数区配置 → 点击「应用」
- 可配置参数：
    - 缩放模式：下拉选择（保持比例/拉伸填充，默认保持比例）
    - 目标尺寸：宽度/高度（输入框，单位px，保持比例时，修改一个值另一个自动同步）
    - 最大限制：宽度/高度最大为10000px，超出时自动限制，状态区提示“图片尺寸已达最大限制”
- UI反馈：缩放后预览区显示新尺寸图片，状态区显示“缩放成功，新尺寸：W×H”

#### （3）特效处理功能
- UI操作路径：左侧工具栏「扩展功能」→ 展开「特效处理」→ 选择子功能（模糊/边缘检测/组合特效）→ 右侧参数区配置 → 点击「应用」
- 各子功能可配置参数：
    - 模糊：下拉选择强度（轻微/强烈，对应不同模糊程度，无需手动输入参数）
    - 边缘检测：下拉选择类型（水平/垂直/全方位，默认全方位）
    - 组合特效：下拉选择预设效果（怀旧/素描/冷色调/暖色调，一键应用，无需额外参数）
- UI反馈：选择特效后实时预览效果，应用成功后状态区显示“XX特效应用成功”

### 3. AI增强功能（UI独立标识，需提前配置）
#### （1）功能前置配置（UI操作）
- 操作路径：顶部菜单栏「帮助」→「参数配置」→ 弹出配置窗口，填写以下参数（必填项标红）：
    1. DeepSeek问答配置：API密钥、服务地址（url）
    2. 豆包图生图配置：ark.api.key、模型ID、请求地址
- UI反馈：配置完成后点击「保存」，状态区显示“配置保存成功”；未配置直接使用AI功能时，弹出提示窗口“请先完成AI参数配置，路径：帮助→参数配置”

#### （2）智能背景移除
- UI操作路径：左侧工具栏「AI功能组」→ 点击「背景移除图标」（人物+透明背景样式）→ 可选配置边缘羽化（0~10px，滑动条调节）→ 点击「开始处理」
- UI反馈：
    - 处理中：预览区显示加载动画，状态区提示“正在智能移除背景，请稍候...”
    - 处理完成：预览区显示透明背景图片，状态区提示“背景移除成功，可直接保存为PNG格式”
    - 处理失败：弹出提示窗口“背景移除失败，请检查图片格式或网络状态”

#### （3）智能修复（内容感知填充）
- UI操作路径：
    1. 左侧工具栏「AI功能组」→ 点击「智能修复图标」（画笔+修复样式）
    2. 预览区：鼠标变为画笔样式，拖拽绘制需要修复的区域（红色虚线框标记）
    3. 右侧参数区：调节画笔大小（5~50px，滑动条）→ 点击「开始处理」
- UI反馈：
    - 绘制区域过大时，状态区提示“修复区域过大，建议缩小范围后重试”
    - 处理完成后，预览区修复区域自动填充，状态区提示“智能修复成功”

#### （4）图片风格化
- UI操作路径：左侧工具栏「AI功能组」→ 点击「风格化图标」（油画样式）→ 右侧参数区配置 → 点击「开始处理」
- 可配置参数：
    - 风格类型：下拉选择（油画/铅笔素描/水彩/卡通，默认油画）
    - 风格强度：0.0~1.0（滑动条+输入框，数值越高风格越明显）
- UI反馈：
    - 处理中：预览区显示加载动画，状态区提示“正在进行风格转换，请稍候...”
    - 处理完成：预览区显示风格化后的图片，状态区提示“风格转换成功，风格类型：XX，强度：XX”

#### （5）超分辨率增强
- UI操作路径：左侧工具栏「AI功能组」→ 点击「超分增强图标」（放大镜+高清样式）→ 右侧参数区配置 → 点击「开始处理」
- 可配置参数：
    - 放大倍数：2倍/4倍（下拉选择，默认2倍）
    - 锐化强度：0.0~1.0（滑动条+输入框，数值越高图片越清晰）
- UI反馈：处理完成后，预览区显示高清图片，状态区提示“超分增强成功，新尺寸：W×H（放大XX倍）”

#### （6）AI辅助色彩增强
- UI操作路径：左侧工具栏「AI功能组」→ 点击「色彩增强图标」（调色盘+AI样式）→ 右侧参数区选择场景（人像/风景/低光，默认自动识别）→ 点击「开始处理」
- UI反馈：无需手动调节参数，AI自动优化色彩，处理完成后状态区提示“AI色彩增强成功，适配场景：XX”

#### （7）AI文字问答与图生图（附加功能）
- AI文字问答：
    1. 操作路径：顶部菜单栏「功能」→「AI问答」→ 弹出问答窗口，输入问题（如“如何优化人像图片色彩”）→ 点击「发送」
    2. UI反馈：回答内容逐字打印显示（模拟打字效果），支持复制回答内容
- 豆包图生图：
    1. 操作路径：顶部菜单栏「功能」→「图生图」→ 弹出生成窗口，输入文本描述（如“一只可爱的猫咪，油画风格”）→ 配置图片尺寸（默认512×512px）→ 点击「生成」
    2. UI反馈：生成中显示加载动画，生成完成后自动显示在预览区，支持保存图片

## 四、UI专属辅助功能（提升用户体验）
1.  图片预览缩放：中央预览区鼠标滚轮向上放大、向下缩小，最大放大至200%，最小缩小至10%
2.  双窗对比模式：预览区右上角点击「对比」按钮，切换单窗/双窗（双窗左侧原图、右侧效果图，便于直观查看差异）
3.  工具栏图标提示：鼠标悬浮在工具栏图标上，显示功能名称+简短操作说明（如“旋转：顺时针旋转图片，支持自定义角度”）
4.  参数重置功能：右侧参数区每个功能下方都有「重置」按钮，点击可恢复该功能的默认参数
5.  批量操作功能：支持批量导入图片，批量应用同一功能（如批量旋转90°、批量添加怀旧特效），操作路径：「编辑」→「批量操作导入」→ 选择功能及参数 → 「批量应用」→ 「批量保存」

| 操作记录为空（撤销/重做） | 对应按钮变灰（不可点击），状态区显示灰色提示文本                            | “已无历史操作可撤销/重做”                                                        |

2.  步骤细化到UI交互：每个功能需明确“点击XX按钮→选择XX选项→输入/调节XX参数→点击XX确认”，附图标样式描述（如“旋转图标：箭头循环样式”），方便用户对照查找
3.  突出UI便捷功能：重点强调“撤销/重做”“双窗对比”“参数重置”“批量操作”等提升效率的功能，明确快捷键（Ctrl+Z/Ctrl+Y）
4.  明确参数范围与限制：在每个功能说明中，标注参数有效范围（如亮度-100%~+100%）、格式限制（如保存透明背景需选PNG），提前规避用户操作错误
5.  补充异常处理说明：单独设立“常见问题”章节，对应UI异常提示，给出具体解决方案（如“提示‘格式错误’怎么办？→ 更换JPG/PNG/BMP格式图片”）
6.  配图参考：建议在用户说明书中插入UI界面截图，标注各操作区域及功能图标，直观展示操作路径

## 七、项目核心特性（UI视角总结）
1.  易用性：所有功能通过可视化UI操作实现，无需代码基础，参数支持滑动条+输入框双重调节，降低操作门槛
2.  实时性：参数调节及功能选择后，预览区实时显示效果，无需等待即可查看结果
3.  容错性：完善的异常提示机制，参数超出范围自动修正，操作错误支持撤销/恢复原图
4.  扩展性：基础功能、扩展功能、AI功能分组清晰，UI支持折叠/展开，兼顾简洁性与功能性
